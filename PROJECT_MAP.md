# 🎓 Mini Project Structure

This project is divided into two main parts:

### 1. 🖥️ **FrontEnd** (Directory: `/Project/LearingPlatform`)
- **Technology**: React + Vite
- **Purpose**: This is the visual part of the application that the user interacts with. 
- **Key Features**: Login screen, Student dashboard, Admin upload panel, Study material viewer.
- **Port**: Runs on `http://localhost:5173`

### 2. ⚙️ **BackEnd** (Directory: `/Project/BackEnd`)
- **Technology**: Node.js + Express + MongoDB
- **Purpose**: This is the "brain" of the application. It handles user authentication, database connections, and file storage.
- **Key Features**: MongoDB connection, JWT Security, Multer file uploads, Database seeding.
- **Port**: Runs on `http://localhost:5000`

---

### 🚀 Quick Start
- To run the **FrontEnd**: Go to `/LearingPlatform` and run `npm run dev`.
- To run the **BackEnd**: Go to `/BackEnd` and run `npm run dev`.
